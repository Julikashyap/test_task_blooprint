Setting Up

First of all you need Python to get started. You can download it here

Once Python is installed, open Terminal and create virtualenv with run python -m virtualenv env_name and then activate this env with the help of command env_name/Script/activate

Once viirtualenv activated then need to install all dependencies, open Terminal and run pip install -r requirements.txt to install all of dependencies. On Virtuale Env env_name

When all dependency is installed successfully then need to run migrate command python manage.py migrate. We have comment out the MySQL DB configuration and used by default DB due to fast setup

Once migration is done then Automatically super user is also created due to we add migration file where adding superuser with username superuser@gmail.com and password Admin@1234

Now you're ready to start to run this server. All you have to do is run python manage.py runserver in Terminal. This will track all the sass file changes and start a local webserver at http://localhost:8000. But you need to see all API documentation then you go to the Swagger

In swagger you get all API which you give me for development.
